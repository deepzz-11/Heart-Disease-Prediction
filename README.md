# Heart Disease Prediction using K-Nearest Neighbors (KNN)

## Project Overview

This project predicts whether a patient has heart disease using the K-Nearest Neighbors (KNN) machine learning algorithm. The project includes data preprocessing, exploratory data analysis, feature engineering, model training, evaluation, and prediction for new patient data.

---

## Dataset

- Dataset: Heart Disease Dataset
- Total Records: 918
- Target Variable: HeartDisease
- Problem Type: Binary Classification

---

## Features

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Data Visualization
- Feature Encoding
- Feature Scaling using StandardScaler
- K-Nearest Neighbors (KNN) Classification
- Model Evaluation
- Heart Disease Prediction for New Patients

---

## Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## Machine Learning Algorithm

- K-Nearest Neighbors (KNN)

---

## Model Performance

**Accuracy:** 88.59%

---

## Repository Contents

- `Heart_Disease_Prediction.ipynb` – Complete project notebook
- `heart.csv` – Dataset used for training and testing

---

## How to Run

1. Clone or download this repository.
2. Open `Heart_Disease_Prediction.ipynb` in Google Colab or Jupyter Notebook.
3. Ensure `heart.csv` is in the same folder as the notebook.
4. Run all cells in order.

---

## Sample Prediction

The model predicts whether a patient is likely to have heart disease based on the following input features:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Maximum Heart Rate
- Exercise-Induced Angina
- Oldpeak
- ST Slope

Possible outputs:

- Heart Disease Detected
- No Heart Disease
